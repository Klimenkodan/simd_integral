Варіант -- 1

Графік залежності знаходиться в папці image

Параметри конфігураційного файлу:
1. relative error -- бажана відносна точність
2. absolute error -- бажана абсолютна точність
3. number of threads -- кількість потоків, за допомогою яких виконувати
4. x1 -- ліва границя по х
5. x2 -- права границя по х
6. y1 -- ліва границя по у
7. y2 -- права границя по у
8. initial steps -- початкова кількість квадратів для розбиття
9. max steps -- максимальна кількість.
